# jlox-rs

A Rust implementation of the tree-walk interpreter in the book [_Crafting Interpreters_](https://github.com/munificent/craftinginterpreters). 

There are bugs to be fixed.

## Usage

```sh
cargo run --bin jlox -- [script]
```
